# Проект, нацеленный на анализ датасета на языке Python

Проект был выполнен в рамках изучения Питона и представляет из себя Jupiter Notebook, содержащий в себе подробный анализ и работу с датасетом, содержащим в себе рейтинг IMDb 1000 фильмов и шоу, выведение гипотезы, работу с гипотезой и выводы на основе полученных данных. 

# Технологии
В процессе выполнения проекта, я использовала различные инструменты и библиотеки Питона для анализа и визуализации данных. К ним относятся:

- "pandas" library (удаление NaN значений, работа с дубликатами, чистка датасета и создание новых столбцов, создание графиков при помощи других библиотек)

- "matplotlib" library (создание графиков)

- "seaborn" library (визуальное оформление графиков)

# Запуск

1. Скопируйте репозиторий:

```bash
git clone https://github.com/sboevadaria/imdb-hypothesis-testing.git
  
cd imdb-hypothesis-testing
```

2. Установите библиотеки:

```bash
pip install pandas matplotlib seaborn
```

3. Скачайте и разместите датасет:

```bash
imdb-hypothesis-testing/

└── dataset/

    └── imdb_top_1000.csv.xls
```

4. Запустите Jupiter Notebook:

```bash
jupyter notebook
```
5. Откройте проект (imdb_hypothesis_testing.ipynb) и запустите все ячейки
